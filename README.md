# Listening Practice
Practice listening of German numbers 

## Getting Started for only practicing 
- download exe directory through [release](https://github.com/chenxshuo/ListeningPractice/releases/tag/0.1)
- unzip the file 
- run the `main.exe`

## Getting Started for development  

### Prerequisites 
- `python 3.6`
- `playsound 1.2.2`

```python
pip install playsound
```
### Installing 
```bash
git clone https://github.com/chenxshuo/ListeningPractice.git
```

### Running 

```bash
cd ListeningPractice
python main.py
```
